In this course, we learnt to use Machine Learning in the design process, and started thinking about the new role and agency of designers in the age of algorithms, data, and fast machines. The course introduced machine learning for designers in three parts: principles of generative learning in its high-level overview, design applications of such methods in the form of projects, and interactive workshop on using machine learning in design. For tangible skills, we learnt how to read machine learning codes and used Python and other open source libraries for designers.

The datasets used for my final project undertaken in this class include:
1. Exterior images from Dwell
2. Swedish Leaf Dataset

Exploring AI and Neural Networks in Design

Project developed as part of the J-Term course at Harvard GSD

Offered by: Gia Jung and Claire Djang

Underlying Principles:
Scraping a Dataset from the Web
Processing and Augmentation of Dataset
Training a Dataset on DCGAN
Creating latent space visualizations
